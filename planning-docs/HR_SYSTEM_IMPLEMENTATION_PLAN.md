## 技术栈规范

### 核心技术
- **框架**: Spring Boot 3.2.5, Spring Data JPA, Spring Security
- **数据库**: PostgreSQL 15, Redis 7.0, RabbitMQ 3.12
- **存储**: MinIO对象存储
- **监控**: Micrometer + Prometheus

### 开发规范
- **编码规范**: 遵循Google Java Style Guide
- **API设计**: RESTful API，OpenAPI 3.0文档
- **数据库**: 遵循数据库命名规范，包含审计字段
- **测试**: JUnit 5, TestContainers, AssertJ
