# 仓库管理系统文档总览

## 1. 子系统概述

仓库管理系统（WAREHOUSE-MANAGEMENT）是GMP系统中负责物料仓储和库存管理的核心子系统。该系统提供全面的仓库运营管理功能，包括物料接收、存储、发放、盘点以及库存优化等，确保物料在GMP环境下的合规管理和高效流转，为生产运营提供可靠的物料保障。

## 4. 技术架构

### 4.1 技术栈
- 后端：Java 17, Spring Boot 3.x, Spring Cloud
- 数据库：PostgreSQL 14, Redis
- 前端：React, TypeScript, Ant Design
- 移动应用：Flutter
- 条码/RFID：Zebra API, RFID SDK
- 集成：RESTful API, WebSocket, Kafka

### 4.2 架构设计
- 分层架构设计
- 微服务架构模式
- 事件驱动集成
- 分布式事务管理
- 高可用集群部署

### 4.3 安全架构
- 基于角色的访问控制
- 操作审计日志
- 敏感数据加密
- 防篡改机制
- 合规性记录保存
