# 中药材检验子系统 - 操作手册

## 文档信息
- **文档版本**: v1.0
- **创建日期**: 2024-01-15
- **最后更新**: 2024-01-15
- **作者**: 系统文档团队
- **审核人**: 质量保证经理、培训经理、GMP合规专家

## 1. 概述

### 1.1 文档目的

本文档为中药材检验子系统的用户操作手册，旨在指导系统用户了解系统功能、掌握操作方法，正确使用中药材检验子系统完成日常工作。本手册适用于系统管理员、质量管理负责人、质量控制主管、检验员、样品管理员等系统用户。

### 1.2 术语定义

| 术语 | 解释说明 |
|-----|---------|
| 样品 | 用于检验的中药材或中药饮片，通常为小批量的代表性样本 |
| 检验计划 | 规定检验项目、检验方法、检验标准和检验周期的计划文件 |
| 检验任务 | 根据检验计划生成的具体工作任务，分配给检验员执行 |
| 检验报告 | 记录检验过程和结果的正式文件，包含检验数据、结论和审核信息 |
| 质量标准 | 规定中药材质量要求的标准文件，包括性状、鉴别、检查、含量测定等项目 |
| 不合格品 | 检验结果不符合质量标准要求的中药材或中药饮片 |
| 电子签名 | 用于确认电子记录真实性和完整性的电子身份验证方式 |
| GMP | 药品生产质量管理规范(Good Manufacturing Practice) |

## 2. 系统登录与导航

### 2.1 系统登录

1. **打开浏览器**：使用支持HTML5的现代浏览器（推荐Chrome、Firefox或Edge最新版本）
2. **访问系统地址**：在浏览器地址栏输入系统URL地址，如：`http://gmp-system.example.com`
3. **输入登录凭证**：
   - 在登录页面输入用户名和密码
   - 如有双因素认证，按照提示完成验证
4. **点击登录**：点击"登录"按钮进入系统
5. **登录后操作**：
   - 首次登录需修改初始密码
   - 查看系统公告和待办事项

### 2.2 系统导航

系统采用顶部菜单和左侧功能菜单相结合的导航方式：

1. **顶部菜单栏**：显示系统名称、用户信息、消息通知和退出按钮
2. **左侧功能菜单**：按照功能模块分类显示各功能入口
3. **主内容区域**：显示当前功能的操作界面
4. **面包屑导航**：显示当前页面的层级路径，方便返回上级页面

### 2.3 常见操作

1. **修改密码**：点击用户头像，选择"修改密码"，输入旧密码和新密码，点击"确定"完成修改
2. **查看通知**：点击顶部菜单的消息图标，查看系统通知和待办任务
3. **退出系统**：点击顶部菜单的"退出"按钮，安全退出系统

## 3. 样品管理

### 3.1 样品接收与登记

1. **进入样品管理模块**：在左侧菜单点击"样品管理"→"样品登记"
2. **填写样品信息**：
   - 基本信息：样品名称、编号、规格、数量、来源
   - 批次信息：批次号、生产日期、有效期
   - 供应商信息：供应商名称、联系人和联系方式
   - 接收信息：接收日期、接收人、检验类型
3. **上传样品图片**：点击"上传图片"，选择样品照片上传
4. **提交样品登记**：点击"保存"按钮完成样品登记
5. **生成样品二维码**：系统自动生成样品唯一二维码，用于样品标识和跟踪

### 3.2 样品查询与跟踪

1. **进入样品查询页面**：在左侧菜单点击"样品管理"→"样品查询"
2. **设置查询条件**：
   - 可按样品编号、名称、批次号、检验状态等条件查询
   - 可设置时间范围和样品来源等高级筛选条件
3. **执行查询**：点击"查询"按钮显示查询结果
4. **查看样品详情**：点击样品列表中的"查看"按钮，查看样品详细信息
5. **跟踪样品状态**：在样品详情页面，可查看样品从接收到检验完成的全过程状态

### 3.3 样品流转管理

1. **样品分派**：
   - 样品管理员在样品详情页面点击"分派样品"
   - 选择接收部门和接收人
   - 填写分派说明，点击"确认分派"
2. **样品退回**：
   - 如样品不符合检验要求，可发起退回操作
   - 在样品详情页面点击"退回样品"
   - 选择退回原因和接收人，点击"确认退回"
3. **样品销毁**：
   - 检验完成后的样品需要销毁时
   - 在样品详情页面点击"销毁样品"
   - 填写销毁记录，包括销毁时间、销毁人和销毁方式
   - 进行电子签名确认

## 4. 检验计划管理

### 4.1 创建检验计划

1. **进入检验计划模块**：在左侧菜单点击"检验计划"→"创建计划"
2. **填写计划基本信息**：
   - 计划名称、编号、制定日期、有效期限
   - 适用范围、检验周期、责任人
3. **添加检验项目**：
   - 点击"添加检验项目"
   - 选择检验类型（外观、鉴别、检查、含量测定等）
   - 指定检验方法和检验标准
   - 设置允收标准和检验数量
4. **关联样品**：选择需要应用此检验计划的样品或样品类型
5. **提交审批**：点击"提交审批"，将检验计划提交给质量控制主管或质量管理负责人审核

### 4.2 检验计划审批

1. **进入审批页面**：质量控制主管或质量管理负责人登录系统，在待办任务中查看待审批的检验计划
2. **审核计划内容**：
   - 查看检验计划的详细信息
   - 检查检验项目设置是否合理
   - 验证检验标准是否符合法规要求
3. **执行审批操作**：
   - 批准：点击"批准"按钮，检验计划生效
   - 驳回：点击"驳回"按钮，填写驳回原因，检验计划返回修改
   - 修改并批准：点击"修改并批准"，修改计划后直接批准
4. **电子签名确认**：审批操作需要进行电子签名，输入密码完成确认

### 4.3 检验计划查询与管理

1. **进入检验计划查询页面**：在左侧菜单点击"检验计划"→"计划查询"
2. **设置查询条件**：
   - 按计划编号、名称、状态等查询
   - 可设置时间范围和责任人等筛选条件
3. **查看计划详情**：点击计划列表中的"查看"按钮，查看计划详细内容
4. **计划操作**：
   - 修订：对于已发布的计划，可点击"修订"创建新版本
   - 暂停：临时暂停计划执行
   - 恢复：恢复已暂停的计划
   - 废止：永久废止不再使用的计划

## 5. 检验任务管理

### 5.1 任务分配

1. **进入任务分配页面**：质量控制主管在左侧菜单点击"检验任务"→"任务分配"
2. **选择待分配任务**：
   - 系统根据检验计划自动生成待分配任务
   - 或手动创建临时检验任务
3. **分配给检验员**：
   - 选择合适的检验员
   - 设置任务优先级和完成期限
   - 添加任务说明和注意事项
4. **确认分配**：点击"确认分配"按钮，任务分配给检验员

### 5.2 任务执行

1. **查看待执行任务**：检验员登录系统，在左侧菜单点击"检验任务"→"待执行任务"
2. **接受任务**：查看任务详情，点击"接受任务"
3. **执行检验操作**：
   - 按照检验计划要求执行各项检验
   - 在系统中录入检验数据和观察结果
   - 上传检验过程中的图片或图谱
4. **记录检验结果**：
   - 对每个检验项目判断合格/不合格
   - 计算检验结果并与标准比对
   - 填写检验结论和异常情况说明
5. **提交检验结果**：点击"提交结果"按钮，将检验数据提交审核

### 5.3 任务监控与查询

1. **进入任务监控页面**：质量控制主管在左侧菜单点击"检验任务"→"任务监控"
2. **查看任务进度**：
   - 监控所有检验任务的当前状态
   - 查看任务完成率和逾期情况
3. **任务调整**：
   - 重新分配任务：将任务从一个检验员转移给另一个检验员
   - 调整优先级：根据紧急程度调整任务优先级
   - 延期处理：对无法按时完成的任务进行延期设置
4. **任务查询**：
   - 设置查询条件，如任务编号、样品名称、检验员等
   - 执行查询，查看历史任务记录
   - 导出任务报表进行分析

## 6. 检验报告管理

### 6.1 报告生成

1. **自动生成报告**：检验结果提交后，系统自动根据检验数据生成检验报告草稿
2. **报告编辑**：
   - 质量控制主管在左侧菜单点击"检验报告"→"报告编辑"
   - 对自动生成的报告进行审核和编辑
   - 补充必要的说明和审核意见
3. **报告格式调整**：
   - 选择报告模板
   - 调整报告布局和内容显示
   - 添加公司logo和相关信息
4. **保存报告草稿**：点击"保存草稿"，保存编辑中的报告

### 6.2 报告审核

1. **提交审核**：报告编辑完成后，点击"提交审核"
2. **审核流程**：
   - 一级审核：质量控制主管审核报告内容的准确性
   - 二级审核：质量管理负责人审核报告的合规性和结论
3. **审核操作**：
   - 批准：审核通过，进入下一审核环节或发布
   - 驳回：审核不通过，返回修改并填写驳回原因
   - 会签：需要其他人员共同审核时，发起会签
4. **电子签名**：每个审核环节都需要审核人进行电子签名确认

### 6.3 报告发布与查询

1. **报告发布**：
   - 最终审核通过后，报告自动发布
   - 发布后的报告不可修改，或修改后需要重新审核
2. **报告查询**：
   - 在左侧菜单点击"检验报告"→"报告查询"
   - 设置查询条件，如报告编号、样品名称、报告日期等
   - 执行查询，查看符合条件的报告
3. **报告操作**：
   - 查看：在线查看报告详细内容
   - 下载：将报告下载为PDF或Excel格式
   - 打印：直接打印报告
   - 导出：批量导出报告数据

## 7. 不合格品管理

### 7.1 不合格品登记

1. **自动触发**：当检验结果不合格时，系统自动触发不合格品登记流程
2. **手动登记**：对于其他途径发现的不合格品，可手动登记
   - 在左侧菜单点击"不合格品管理"→"不合格品登记"
   - 填写不合格品信息，包括样品信息、不合格项目、不合格程度
   - 上传相关证据图片或文件
3. **提交评审**：点击"提交评审"按钮，进入不合格品评审流程

### 7.2 不合格品评审

1. **评审会议准备**：
   - 质量管理负责人组织不合格品评审会议
   - 系统发送评审通知给相关人员
   - 准备评审所需的检验数据和相关信息
2. **执行评审**：
   - 评审人员在左侧菜单点击"不合格品管理"→"不合格品评审"
   - 查看不合格品详细信息和检验数据
   - 提出评审意见和处理建议
3. **评审结论**：
   - 返工/重新检验：返回生产部门返工后重新检验
   - 降级使用：降低等级后使用
   - 销毁：直接销毁不合格品
   - 让步接收：特殊情况下批准让步接收
4. **记录评审结果**：填写评审结论、处理方案和执行期限

### 7.3 不合格品处理跟踪

1. **处理任务分配**：根据评审结论，分配相应的处理任务给责任人
2. **处理过程跟踪**：
   - 在左侧菜单点击"不合格品管理"→"处理跟踪"
   - 监控不合格品处理进度
   - 记录处理过程中的关键节点和数据
3. **处理结果确认**：
   - 处理完成后，执行人提交处理结果
   - 质量管理负责人确认处理结果的有效性
   - 记录最终处理结论和验证数据
4. **不合格品统计分析**：
   - 生成不合格品统计报表
   - 分析不合格品趋势和根本原因
   - 支持质量改进决策

## 8. 质量标准管理

### 8.1 标准录入与维护

1. **进入标准管理页面**：在左侧菜单点击"质量标准"→"标准维护"
2. **创建新标准**：
   - 点击"创建标准"按钮
   - 填写标准基本信息：标准名称、标准编号、发布日期、实施日期
   - 选择标准类型：国家标准、行业标准、企业标准
3. **编辑标准内容**：
   - 添加检验项目：性状、鉴别、检查、含量测定等
   - 设置每个项目的检验方法、限量要求和允差范围
   - 上传标准文本或相关参考资料
4. **保存与审核**：点击"保存"按钮，提交标准进行审核

### 8.2 标准审核与发布

1. **标准审核**：
   - 质量标准起草完成后，提交给质量管理负责人审核
   - 审核内容包括标准的合规性、合理性和可操作性
   - 审核通过后进入发布流程
2. **标准发布**：
   - 审核通过的标准，由授权人员进行发布
   - 发布后的标准生效，可用于检验活动
   - 系统自动通知相关人员标准已更新

### 8.3 标准查询与版本管理

1. **标准查询**：
   - 在左侧菜单点击"质量标准"→"标准查询"
   - 设置查询条件：标准名称、编号、类型等
   - 执行查询，查看符合条件的标准
2. **版本管理**：
   - 查看标准的所有历史版本
   - 比较不同版本之间的差异
   - 查看标准的修订记录和原因
3. **标准状态管理**：
   - 有效：当前正在使用的标准
   - 过期：已过实施有效期的标准
   - 废止：不再使用的标准
   - 草稿：正在起草中的标准

## 9. 数据统计与分析

### 9.1 检验数据统计

1. **进入统计分析模块**：在左侧菜单点击"统计分析"→"检验数据统计"
2. **选择统计维度**：
   - 按样品类型统计
   - 按检验项目统计
   - 按时间周期统计
   - 按检验员统计
3. **设置统计条件**：
   - 选择统计时间范围
   - 选择需要统计的样品范围
   - 设置其他筛选条件
4. **生成统计图表**：
   - 系统自动生成柱状图、饼图、趋势图等统计图表
   - 可切换不同图表类型查看数据
   - 可导出图表为图片或PDF格式

### 9.2 质量趋势分析

1. **进入趋势分析页面**：在左侧菜单点击"统计分析"→"质量趋势分析"
2. **选择分析项目**：
   - 选择需要分析的关键质量指标
   - 设置分析周期（日、周、月、季度、年）
3. **生成趋势图表**：
   - 系统生成质量指标随时间变化的趋势图
   - 显示警戒限和行动限，帮助识别异常波动
4. **趋势解读**：
   - 分析质量指标的稳定性和变化趋势
   - 识别潜在的质量问题和改进机会
   - 支持质量决策和持续改进

### 9.3 报表生成

1. **进入报表中心**：在左侧菜单点击"统计分析"→"报表中心"
2. **选择报表类型**：
   - 检验日报表
   - 检验月报表
   - 不合格品统计报表
   - 检验效率报表
3. **设置报表参数**：
   - 选择报表时间范围
   - 选择需要包含的数据范围
   - 设置报表格式和显示选项
4. **生成并导出报表**：
   - 点击"生成报表"按钮，系统自动生成报表
   - 可预览报表内容
   - 导出为Excel、PDF等格式
   - 支持设置定时自动生成和发送报表

## 10. 系统管理

### 10.1 用户管理

1. **进入用户管理页面**：系统管理员在左侧菜单点击"系统管理"→"用户管理"
2. **创建用户**：
   - 点击"创建用户"按钮
   - 填写用户基本信息：用户名、姓名、部门、职位、联系方式
   - 设置初始密码和角色权限
3. **用户维护**：
   - 编辑用户信息：修改用户基本信息和权限
   - 重置密码：当用户忘记密码时，重置为初始密码
   - 启用/禁用用户：控制用户账户状态
   - 删除用户：删除不再使用的用户账户

### 10.2 角色与权限管理

1. **进入角色管理页面**：系统管理员在左侧菜单点击"系统管理"→"角色管理"

## 11. 常见问题与故障处理

### 11.1 登录问题

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|--------|
| 无法登录系统 | 用户名或密码错误 | 确认用户名和密码是否正确，区分大小写 |
| 无法登录系统 | 账户被锁定 | 联系系统管理员解锁账户，检查登录尝试次数 |
| 无法登录系统 | 网络连接问题 | 检查网络连接，确认系统是否正常运行 |
| 登录后页面空白 | 浏览器兼容性问题 | 升级浏览器至最新版本，或尝试其他浏览器 |

### 11.2 数据操作问题

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|--------|
| 无法保存数据 | 缺少必要字段或字段格式错误 | 检查并补充必要信息，确保格式正确 |
| 无法保存数据 | 权限不足 | 联系系统管理员确认用户权限 |
| 数据查询无结果 | 查询条件设置错误 | 调整查询条件，尝试更宽泛的搜索条件 |
| 数据显示异常 | 数据加载未完成 | 等待数据加载完成，或刷新页面重试 |

### 11.3 系统性能问题

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|--------|
| 页面响应缓慢 | 网络带宽不足 | 检查网络连接，减少同时进行的操作 |
| 页面响应缓慢 | 系统负载过高 | 避开系统高峰期，或联系管理员优化系统 |
| 文件上传失败 | 文件大小超限 | 检查文件大小，压缩或分割大文件 |
| 文件上传失败 | 浏览器兼容性问题 | 更换浏览器或升级至最新版本 |
