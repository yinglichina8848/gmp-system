## 1️⃣ 模板库概述

本提示词模板库为GMP环境下人事管理子系统提供标准化的提示词模板集合。所有模板均遵循GMP合规性要求，确保系统在各个功能模块中提供清晰、专业、一致的用户提示。开发人员和产品设计人员可以直接使用或基于这些模板进行定制，以确保系统提示词的质量和合规性。

### 1.1 模板分类

模板库按以下方式进行分类：

- **按提示类型分类**：信息提示、警告提示、错误提示、确认提示、帮助提示、GMP合规性提示
- **按功能模块分类**：系统登录、组织架构管理、员工信息管理、考勤管理、培训管理、资质证书管理、GMP合规性管理、系统管理
- **按重要程度分类**：常规提示、重要提示、关键GMP提示

### 1.2 模板使用说明

使用本模板库时，请遵循以下原则：

1. **选择合适的模板**：根据提示类型和功能模块选择最匹配的模板
2. **自定义变量内容**：将模板中的占位符替换为实际的变量内容
3. **保持风格一致**：在自定义过程中保持语言风格和格式的一致性
4. **验证合规性**：确保自定义后的提示词符合GMP合规性要求
5. **记录变更**：记录模板使用和自定义的历史，以便追溯

## 4️⃣ 提示词模板自定义指南

### 4.1 模板变量说明

在使用模板时，需要将以下类型的变量替换为实际内容：

- **[操作名称]**：具体的操作动词和对象，如「保存员工信息」
- **[项目名称]**：具体的项目或记录名称，如「GMP设备操作培训」
- **[员工姓名]**：实际员工的姓名
- **[日期]**：具体的日期，格式为YYYY-MM-DD
- **[时间]**：具体的时间，格式为HH:MM:SS
- **[数量]**：具体的数字，如「25」
- **[原因]**：具体的原因说明
- **[建议]**：具体的操作建议

### 4.2 自定义原则

在自定义提示词时，应遵循以下原则：

1. **保持核心信息**：确保提示词包含所有必要的信息点
2. **个性化内容**：根据具体场景和用户角色调整内容
3. **保持合规性**：确保所有提示词符合GMP要求
4. **语言一致性**：使用与系统其他部分一致的语言风格
5. **简明扼要**：避免冗长的描述，突出重点

### 4.3 自定义示例

以"警告提示模板"为例，自定义前后的对比：

**原始模板**：
```
警告
您即将执行的操作可能导致[潜在风险]。
建议：[建议采取的措施]

按钮：继续操作 / 取消
```

**自定义后**：
```
警告
您即将修改张三的GMP关键岗位资质信息，此操作可能影响其GMP合规状态评估。
建议：确认新信息的准确性，确保有适当的审批文件，并通知相关监管人员。

按钮：继续操作 / 取消
```

## 5️⃣ 提示词效果评估

### 5.1 评估标准

使用以下标准评估提示词的有效性：

1. **清晰度**：提示词是否清晰明了，易于理解
2. **完整性**：是否包含所有必要的信息
3. **合规性**：是否符合GMP要求和相关法规
4. **有用性**：是否帮助用户做出正确决策
5. **一致性**：与系统其他部分的提示词风格是否一致

### 5.2 收集反馈

定期收集用户对提示词的反馈：

- 通过用户测试收集直接反馈
- 分析系统使用日志，了解提示词的实际效果
- 定期进行用户满意度调查
- 收集技术支持团队收到的相关问题

### 5.3 持续优化

根据评估结果和反馈持续优化提示词：

- 更新过时或不准确的提示词
- 优化表述方式，提高清晰度
- 根据新的GMP要求更新相关提示
- 添加新的场景和功能模块的提示词
