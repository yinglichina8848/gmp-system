## 1️⃣ 欢迎使用人事子系统

欢迎使用GMP环境下人事管理子系统！本快速上手指南将帮助您快速了解系统的基本功能和使用方法，让您能够尽快上手操作。

### 1.1 系统简介

GMP环境下人事管理子系统是专为制药企业设计的人力资源管理解决方案，旨在满足制药行业严格的GMP合规性要求，同时提供高效、便捷的人事管理功能。

### 1.2 主要功能

- **组织架构管理**：管理企业部门、岗位等组织结构
- **员工信息管理**：维护员工基本信息和档案资料
- **考勤管理**：管理员工打卡、请假、排班等
- **培训管理**：组织和记录员工培训活动
- **资质证书管理**：管理员工GMP相关资质证书
- **GMP合规性管理**：提供权限控制、审计跟踪等GMP专用功能

## 2️⃣ 首次登录系统

### 2.1 登录准备

在使用系统前，您需要：

1. 确保您的电脑已连接互联网
2. 使用推荐的浏览器（Chrome、Firefox、Edge或Safari的最新版本）
3. 获得您的用户名和初始密码（由系统管理员提供）

### 2.2 系统登录

1. 打开浏览器，输入系统URL地址（例如：`http://[服务器IP]:8080/gmp-hr`）
2. 在登录界面输入您的用户名和初始密码
3. 输入验证码（如有）
4. 点击「登录」按钮

### 2.3 修改初始密码

首次登录成功后，系统会提示您修改初始密码。请按照以下步骤操作：

1. 输入当前密码（初始密码）
2. 设置新密码（密码应包含大小写字母、数字和特殊字符，长度不少于8位）
3. 确认新密码
4. 点击「保存」按钮

## 3️⃣ 系统界面导航

### 3.1 界面布局概览

系统主界面分为三个主要区域：

1. **顶部导航栏**：显示系统名称、当前用户信息、通知中心等
2. **左侧菜单栏**：显示系统功能模块和子菜单
3. **主内容区**：显示当前操作的功能页面内容

### 3.2 顶部导航栏功能

- **用户头像**：点击可访问个人设置、修改密码等
- **通知中心**：显示系统通知和待办事项
- **帮助按钮**：获取系统使用帮助
- **退出按钮**：安全退出系统

### 3.3 左侧菜单栏使用

- 点击模块名称可展开/折叠子菜单
- 点击菜单项可切换到相应的功能页面
- 菜单项前的图标表示不同功能类别

## 4️⃣ 基础功能操作

### 4.1 搜索与筛选

大多数列表页面都提供了搜索和筛选功能：

1. **搜索框**：输入关键词快速查找内容
2. **筛选条件**：点击筛选按钮设置多个筛选条件
3. **高级搜索**：点击高级搜索按钮进行更精确的查找

### 4.2 新增记录

在各模块列表页面：

1. 点击「新增」或「添加」按钮
2. 在弹出的表单中填写相关信息
3. 上传必要的附件（如有）
4. 点击「保存」或「确定」按钮

### 4.3 编辑记录

1. 在列表中找到需要编辑的记录
2. 点击操作列的「编辑」按钮
3. 修改相关信息
4. 点击「保存」或「确定」按钮

### 4.4 删除记录

1. 在列表中找到需要删除的记录
2. 点击操作列的「删除」按钮
3. 在弹出的确认对话框中点击「确定」

### 4.5 导出数据

1. 在列表页面设置好筛选条件
2. 点击「导出」按钮
3. 选择导出格式（通常为Excel）
4. 系统生成并下载文件

## 5️⃣ 常用功能快速入门

### 5.1 员工信息查询

1. 从左侧菜单选择「员工信息管理」→「员工列表」
2. 在搜索框中输入员工姓名或工号
3. 点击「搜索」按钮
4. 在列表中查看员工基本信息
5. 点击员工姓名可查看详细信息

### 5.2 请假申请

1. 从左侧菜单选择「考勤管理」→「请假管理」→「请假申请」
2. 点击「新增申请」按钮
3. 选择请假类型（年假/病假/事假等）
4. 设置请假起止时间
5. 填写请假原因
6. 上传相关证明材料（如病假证明）
7. 点击「提交」按钮
8. 等待审批结果（可在「请假记录」中查看进度）

### 5.3 查看个人考勤

1. 从左侧菜单选择「考勤管理」→「考勤统计」→「个人考勤统计」
2. 选择查询月份
3. 点击「查询」按钮
4. 查看个人考勤统计信息，包括出勤、迟到、请假等记录

### 5.4 证书上传

1. 从左侧菜单选择「资质证书管理」→「证书管理」
2. 点击「上传证书」按钮
3. 填写证书基本信息（持有人、类型、编号、有效期等）
4. 上传证书电子版
5. 点击「确定」按钮提交
6. 等待审核通过

### 5.5 查看系统通知

1. 点击顶部导航栏的通知图标
2. 在下拉列表中查看最新通知
3. 点击通知标题可查看详情

## 6️⃣ GMP合规操作指南

### 6.1 电子签名使用

在需要进行电子签名确认的操作页面：

1. 点击「电子签名」或「确认签名」按钮
2. 在弹出的对话框中输入您的用户名和密码
3. 点击「确认签名」按钮完成签名

### 6.2 查看个人操作日志

1. 从左侧菜单选择「GMP合规性管理」→「审计日志」
2. 在搜索框中输入您的用户名
3. 设置日期范围
4. 点击「查询」按钮
5. 查看您的操作记录

## 7️⃣ 快捷键使用

系统支持以下快捷键操作，提高工作效率：

| 快捷键 | 功能 |
|-------|------|
| Ctrl + F | 在当前页面搜索 |
| Ctrl + S | 保存当前表单 |
| Alt + N | 新增记录 |
| Alt + E | 编辑选中记录 |
| Alt + D | 删除选中记录 |
| Alt + R | 刷新当前页面 |

## 8️⃣ 常见问题解答

### 8.1 登录问题

**Q: 忘记密码怎么办？**
A: 点击登录页面的「忘记密码」链接，按照提示进行密码重置。

**Q: 无法登录系统怎么办？**
A: 请检查网络连接、浏览器版本，确认用户名和密码是否正确。如问题持续，请联系系统管理员。

### 8.2 功能使用问题

**Q: 如何查看我的请假记录？**
A: 从左侧菜单选择「考勤管理」→「请假管理」→「请假记录」即可查看。

**Q: 如何更新我的个人信息？**
A: 点击顶部导航栏的用户头像，选择「个人设置」，修改相关信息后保存。

### 8.3 系统性能问题

**Q: 系统运行缓慢怎么办？**
A: 请尝试清除浏览器缓存，关闭不必要的标签页，或更换推荐的浏览器。

**Q: 页面显示异常怎么办？**
A: 刷新页面或尝试使用不同的浏览器。如问题持续，请联系系统管理员。

## 9️⃣ 获得帮助

如果您在使用系统过程中遇到问题或需要帮助，请通过以下方式获取支持：
