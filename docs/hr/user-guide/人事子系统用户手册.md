## 1️⃣ 系统概述

### 1.1 系统简介

GMP环境下人事管理子系统是专为制药企业设计的人力资源管理解决方案，旨在满足制药行业严格的GMP合规性要求，同时提供高效、便捷的人事管理功能。系统采用现代化的微服务架构和前后端分离技术，确保了系统的稳定性、安全性和良好的用户体验。

### 1.2 系统功能

系统主要功能模块包括：

- **组织架构管理**：管理企业部门、岗位等组织结构信息
- **员工信息管理**：维护员工基本信息、档案资料等
- **考勤管理**：管理员工打卡、请假、排班等考勤相关业务
- **培训管理**：组织和记录员工培训活动
- **资质证书管理**：管理员工GMP相关资质证书
- **GMP合规性管理**：提供GMP专用的权限控制、审计跟踪等功能

### 1.3 系统优势

- **GMP合规性**：完全符合制药行业GMP对计算机化系统的要求
- **模块化设计**：功能模块独立，易于使用和管理
- **用户友好界面**：直观的操作界面，降低学习成本
- **安全可靠**：多层次的安全机制，确保数据安全
- **灵活配置**：支持多种业务场景的灵活配置

### 1.4 浏览器要求

系统推荐使用以下浏览器：

- Google Chrome (推荐版本: 90+)
- Mozilla Firefox (推荐版本: 88+)
- Microsoft Edge (推荐版本: 90+)
- Apple Safari (推荐版本: 14+)

为获得最佳体验，建议使用最新版本的浏览器，并确保JavaScript和Cookie已启用。

## 2️⃣ 系统登录

### 2.1 登录界面

1. 在浏览器中输入系统URL地址（例如：`http://[服务器IP]:8080/gmp-hr`）
2. 系统将显示登录界面

### 2.2 账号登录

1. 在登录界面输入您的用户名和密码
2. 输入验证码（如有）
3. 点击「登录」按钮

### 2.3 登录失败处理

如果登录失败，系统会显示相应的错误提示。常见的登录失败原因包括：

- 用户名或密码错误
- 验证码输入错误
- 账号被锁定
- 账号过期
- 网络连接问题

如遇登录问题，请联系系统管理员获取帮助。

### 2.4 密码找回

1. 在登录界面点击「忘记密码」链接
2. 按照提示输入用户名或邮箱
3. 系统将发送验证码或重置链接到您的邮箱
4. 按照邮件中的指引重置密码

### 2.5 安全注意事项

- 首次登录后，请立即修改初始密码
- 定期更换密码，建议每90天更换一次
- 密码应包含大小写字母、数字和特殊字符
- 不要与他人共享您的账号和密码
- 登录系统后，离开电脑时请锁定屏幕或退出系统
- 不要在公共电脑上登录系统，如必须使用，请确保退出后清除浏览器缓存

## 3️⃣ 系统主界面

### 3.1 界面布局

系统主界面采用经典的三段式布局：

1. **顶部导航栏**：显示系统名称、当前用户信息、通知、帮助等
2. **左侧菜单栏**：显示系统功能模块和子菜单
3. **主内容区**：显示当前操作的功能页面

### 3.2 顶部导航栏

顶部导航栏包含以下内容：

- **系统Logo**：显示系统标志
- **系统名称**：显示当前系统名称
- **通知中心**：显示系统通知和消息
- **用户信息**：显示当前登录用户名和头像
- **快速操作**：提供快速访问的功能（如个人设置、密码修改等）
- **退出系统**：点击退出当前登录

### 3.3 左侧菜单栏

左侧菜单栏包含系统的主要功能模块，点击模块名称可展开子菜单，选择具体功能进行操作。菜单支持折叠和展开，方便用户使用。

### 3.4 主内容区

主内容区根据用户选择的功能模块显示相应的页面内容。主要包含：

- **页面标题**：显示当前页面的功能名称
- **功能操作区**：提供新增、编辑、删除等操作按钮
- **数据展示区**：以表格、图表等形式展示数据
- **搜索筛选区**：提供数据查询和筛选功能

### 3.5 快捷操作

系统提供以下快捷操作：

- **刷新页面**：更新当前页面的数据
- **全屏显示**：将当前页面切换为全屏模式
- **帮助文档**：打开当前功能的帮助说明
- **意见反馈**：提交使用问题和建议

## 4️⃣ 组织架构管理

### 4.1 部门管理

#### 4.1.1 部门列表

1. 从左侧菜单选择「组织架构管理」→「部门管理」
2. 页面显示当前系统中的部门列表
3. 可通过搜索框搜索部门名称
4. 可通过筛选条件筛选部门状态

#### 4.1.2 新增部门

1. 在部门列表页面点击「新增」按钮
2. 在弹出的对话框中填写部门信息：
   - 部门名称
   - 部门编码
   - 上级部门
   - 部门负责人
   - 联系电话
   - 部门描述
3. 点击「确定」按钮保存部门信息

#### 4.1.3 编辑部门

1. 在部门列表页面找到需要编辑的部门
2. 点击操作列的「编辑」按钮
3. 在弹出的对话框中修改部门信息
4. 点击「确定」按钮保存修改

#### 4.1.4 删除部门

1. 在部门列表页面找到需要删除的部门
2. 点击操作列的「删除」按钮
3. 在弹出的确认对话框中点击「确定」

**注意**：删除部门前请确保该部门下没有员工，否则无法删除。如需删除含有员工的部门，请先将员工转移到其他部门。

#### 4.1.5 查看部门详情

1. 在部门列表页面找到需要查看的部门
2. 点击部门名称或操作列的「查看」按钮
3. 页面显示部门的详细信息，包括部门基本信息和部门下的员工列表

### 4.2 岗位管理

#### 4.2.1 岗位列表

1. 从左侧菜单选择「组织架构管理」→「岗位管理」
2. 页面显示当前系统中的岗位列表
3. 可通过搜索框搜索岗位名称
4. 可通过筛选条件筛选岗位状态和所属部门

#### 4.2.2 新增岗位

1. 在岗位列表页面点击「新增」按钮
2. 在弹出的对话框中填写岗位信息：
   - 岗位名称
   - 岗位编码
   - 所属部门
   - 岗位级别
   - 岗位描述
3. 点击「确定」按钮保存岗位信息

#### 4.2.3 编辑岗位

1. 在岗位列表页面找到需要编辑的岗位
2. 点击操作列的「编辑」按钮
3. 在弹出的对话框中修改岗位信息
4. 点击「确定」按钮保存修改

#### 4.2.4 删除岗位

1. 在岗位列表页面找到需要删除的岗位
2. 点击操作列的「删除」按钮
3. 在弹出的确认对话框中点击「确定」

**注意**：删除岗位前请确保没有员工担任该岗位，否则无法删除。如需删除有员工担任的岗位，请先将员工调整到其他岗位。

## 5️⃣ 员工信息管理

### 5.1 员工列表

1. 从左侧菜单选择「员工信息管理」→「员工列表」
2. 页面显示当前系统中的员工列表
3. 可通过搜索框搜索员工姓名、工号等信息
4. 可通过筛选条件筛选员工状态、所属部门、岗位等
5. 可点击列头进行排序

### 5.2 新增员工

1. 在员工列表页面点击「新增」按钮
2. 系统打开员工信息录入页面，填写以下信息：
   - 基本信息：姓名、性别、出生日期、身份证号等
   - 工作信息：工号、入职日期、所属部门、岗位等
   - 联系方式：手机号码、邮箱地址等
   - 教育经历：学历、毕业院校、专业等
   - 工作经历：过往工作经验等
3. 上传相关附件（如身份证复印件、学历证书等）
4. 点击「保存」按钮提交员工信息

### 5.3 编辑员工信息

1. 在员工列表页面找到需要编辑的员工
2. 点击操作列的「编辑」按钮
3. 在弹出的编辑页面中修改员工信息
4. 点击「保存」按钮保存修改

### 5.4 查看员工详情

1. 在员工列表页面找到需要查看的员工
2. 点击操作列的「详情」按钮或员工姓名
3. 页面显示员工的详细信息，包括个人基本信息、工作信息、培训记录、证书信息等

### 5.5 员工离职管理

1. 在员工列表页面找到需要办理离职的员工
2. 点击操作列的「离职」按钮
3. 在弹出的对话框中填写离职信息：
   - 离职日期
   - 离职原因
   - 离职类型
4. 点击「确定」按钮提交离职申请

### 5.6 员工批量导入

1. 在员工列表页面点击「批量导入」按钮
2. 下载员工信息导入模板
3. 按照模板格式填写员工信息
4. 上传填写好的Excel文件
5. 系统进行数据校验并导入员工信息

### 5.7 员工信息导出

1. 在员工列表页面设置好筛选条件
2. 点击「导出」按钮
3. 选择导出范围（全部数据或当前筛选结果）
4. 系统生成并下载Excel格式的员工信息报表

## 6️⃣ 考勤管理

### 6.1 考勤设置

#### 6.1.1 考勤规则设置

1. 从左侧菜单选择「考勤管理」→「考勤设置」→「考勤规则设置」
2. 页面显示当前系统中的考勤规则列表
3. 点击「新增」按钮创建新的考勤规则，填写以下信息：
   - 规则名称
   - 工作天数/周
   - 工作时间
   - 迟到/早退判定标准
   - 缺勤判定标准
   - 加班计算方式
4. 点击「保存」按钮保存规则设置

#### 6.1.2 排班设置

1. 从左侧菜单选择「考勤管理」→「考勤设置」→「排班设置」
2. 选择部门和月份
3. 为员工分配工作班次
4. 可通过模板快速排班
5. 点击「保存」按钮保存排班信息

### 6.2 打卡记录

#### 6.2.1 打卡记录查询

1. 从左侧菜单选择「考勤管理」→「打卡记录」
2. 设置查询条件（如日期范围、部门、员工等）
3. 点击「查询」按钮查看打卡记录
4. 可查看员工的打卡时间、打卡地点等信息

#### 6.2.2 补卡申请

1. 在打卡记录页面找到需要补卡的员工和日期
2. 点击操作列的「补卡」按钮
3. 在弹出的对话框中填写补卡原因、实际打卡时间等信息
4. 上传相关证明材料（如照片）
5. 点击「提交」按钮
6. 等待管理员审批

### 6.3 请假管理

#### 6.3.1 请假申请

1. 从左侧菜单选择「考勤管理」→「请假管理」→「请假申请」
2. 点击「新增申请」按钮
3. 填写请假信息：
   - 请假类型（年假、病假、事假等）
   - 请假时间（开始时间、结束时间）
   - 请假原因
   - 工作交接人
4. 上传相关证明材料（如病假证明）
5. 点击「提交」按钮
6. 等待审批

#### 6.3.2 请假审批

1. 从左侧菜单选择「考勤管理」→「请假管理」→「请假审批」
2. 页面显示待审批的请假申请列表
3. 点击操作列的「审批」按钮
4. 查看请假详情和相关证明材料
5. 选择「同意」或「拒绝」，并填写审批意见
6. 点击「提交」按钮完成审批

#### 6.3.3 请假记录查询

1. 从左侧菜单选择「考勤管理」→「请假管理」→「请假记录」
2. 设置查询条件（如日期范围、部门、员工、请假状态等）
3. 点击「查询」按钮查看请假记录

### 6.4 考勤统计

#### 6.4.1 个人考勤统计

1. 从左侧菜单选择「考勤管理」→「考勤统计」→「个人考勤统计」
2. 选择月份和员工
3. 点击「查询」按钮
4. 页面显示员工在指定月份的考勤统计信息，包括：
   - 出勤天数
   - 迟到/早退次数
   - 缺勤天数
   - 请假天数
   - 加班时长

#### 6.4.2 部门考勤统计

1. 从左侧菜单选择「考勤管理」→「考勤统计」→「部门考勤统计」
2. 选择月份和部门
3. 点击「查询」按钮
4. 页面显示部门在指定月份的考勤统计概览和部门内员工的考勤明细

#### 6.4.3 考勤报表导出

1. 在考勤统计页面设置好查询条件
2. 点击「导出报表」按钮
3. 系统生成并下载Excel格式的考勤报表

#### 7.4.2 颁发证书

4. 点击「确定」按钮颁发证书

### 8.2 证书管理

#### 8.2.1 证书列表

1. 从左侧菜单选择「资质证书管理」→「证书管理」
2. 页面显示当前系统中的证书列表
3. 可通过搜索框搜索证书名称或持有人
4. 可通过筛选条件筛选证书类型、状态、有效期等

#### 8.2.2 上传证书

1. 在证书列表页面点击「上传证书」按钮
2. 填写证书信息：
   - 持有人（员工姓名）
   - 证书类型
   - 证书编号
   - 发证日期
   - 有效期至
   - 发证机构
3. 上传证书电子版（支持PDF、JPG、PNG格式）
4. 点击「确定」按钮提交证书信息

#### 8.2.3 证书审核

1. 从左侧菜单选择「资质证书管理」→「证书审核」

系统会自动检测证书的有效期，并在证书到期前30天、15天、7天发送提醒通知。用户可在系统首页或通知中心查看即将到期的证书信息。

## 1️⃣1️⃣ 常见问题解答

### 11.1 登录问题

**Q: 忘记密码怎么办？**
A: 点击登录页面的「忘记密码」链接，按照提示进行密码重置。

**Q: 账号被锁定怎么处理？**
A: 账号被锁定通常是由于多次登录失败导致。请联系系统管理员解锁账号。

### 11.2 数据操作问题

**Q: 如何快速导入大量员工信息？**
A: 在员工列表页面点击「批量导入」按钮，下载模板并填写信息后上传即可。

**Q: 如何导出考勤报表？**
A: 在考勤统计页面设置查询条件后，点击「导出报表」按钮即可。

### 11.3 证书管理问题

**Q: 如何查看即将到期的证书？**
A: 系统会自动发送证书到期提醒。您也可以在证书统计页面查看即将到期的证书列表。

**Q: 证书审核需要多长时间？**
A: 通常情况下，证书审核会在3个工作日内完成。

### 11.4 GMP合规问题

**Q: 如何查看操作审计日志？**
A: 从左侧菜单选择「GMP合规性管理」→「审计日志」，设置查询条件后即可查看。

**Q: 电子签名的有效期是多久？**
A: 电子签名的有效期为当前操作会话期间，通常为30分钟。

### 11.5 系统使用问题

**Q: 系统运行缓慢怎么办？**
A: 请尝试清除浏览器缓存，或使用系统推荐的浏览器。如问题持续，请联系系统管理员。

**Q: 如何获取更多的帮助？**
A: 您可以点击页面右上角的「帮助」按钮查看在线帮助文档，或联系系统管理员获取支持。
