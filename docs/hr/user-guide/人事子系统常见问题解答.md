## 1️⃣ 登录与账号问题

### 1.1 无法登录系统

**问题**：输入用户名和密码后无法登录系统，页面没有任何提示或提示错误。

**解决方案**：
1. 检查网络连接是否正常
2. 确认您输入的用户名和密码是否正确（注意大小写）
3. 尝试清除浏览器缓存和Cookie后重新登录
4. 确认您的浏览器是否为系统推荐的版本
5. 如以上方法无效，请联系系统管理员确认账号状态

### 1.2 忘记密码

**问题**：忘记了系统登录密码，无法登录。

**解决方案**：
1. 在登录页面点击「忘记密码」链接
2. 输入您的用户名或注册邮箱
3. 按照系统提示完成身份验证
4. 系统将发送密码重置链接或验证码到您的邮箱
5. 按照邮件指引设置新密码

### 1.3 账号被锁定

**问题**：多次输入错误密码后，账号被锁定。

**解决方案**：
1. 账号锁定通常是为了安全起见，在多次登录失败后自动触发
2. 锁定时间一般为30分钟，之后可自动解锁
3. 如需要立即解锁，请联系系统管理员
4. 重置密码后账号会自动解锁

### 1.4 修改密码

**问题**：想要修改当前登录密码。

**解决方案**：
1. 登录系统后，点击顶部导航栏的用户头像
2. 选择「修改密码」选项
3. 输入当前密码、新密码和确认新密码
4. 点击「保存」按钮完成修改
5. 密码要求：至少8位，包含大小写字母、数字和特殊字符

## 2️⃣ 基础操作问题

### 2.1 页面加载缓慢

**问题**：系统页面加载时间较长，响应缓慢。

**解决方案**：
1. 检查网络连接速度
2. 关闭浏览器中不必要的标签页和应用
3. 清除浏览器缓存和Cookie
4. 更新浏览器至最新版本
5. 如使用VPN，尝试暂时关闭VPN后再访问
6. 如问题持续，请联系系统管理员检查服务器状态

### 2.2 无法保存数据

**问题**：填写表单后点击保存按钮，数据无法保存成功。

**解决方案**：
1. 检查必填项是否已全部填写
2. 确认输入内容是否符合系统要求（如日期格式、数值范围等）
3. 检查网络连接是否稳定
4. 刷新页面后重新尝试
5. 如问题持续，请联系系统管理员并提供具体错误信息

### 2.3 文件上传失败

**问题**：尝试上传文件（如员工照片、证书扫描件等）时失败。

**解决方案**：
1. 确认文件格式是否符合系统要求（通常支持JPG、PNG、PDF格式）
2. 检查文件大小是否超过系统限制（通常单个文件不超过10MB）
3. 检查网络连接是否稳定
4. 尝试压缩文件或调整文件格式后重新上传
5. 如问题持续，请联系系统管理员

### 2.4 打印功能异常

**问题**：点击打印按钮后，页面显示异常或无法正常打印。

**解决方案**：
1. 使用浏览器的打印功能（Ctrl+P）替代系统打印按钮
2. 调整浏览器打印设置，如纸张大小、方向等
3. 尝试将页面保存为PDF后再打印
4. 更新浏览器至最新版本

## 3️⃣ 员工信息管理问题

### 3.1 无法新增员工

**问题**：在员工信息管理模块，点击「新增」按钮后无法正常添加员工信息。

**解决方案**：
1. 确认您是否有员工信息新增的权限
2. 检查必填项是否已全部填写，特别是身份证号、工号等关键字段
3. 确认身份证号和工号是否已被其他员工使用
4. 检查网络连接，刷新页面后重新尝试

### 3.2 员工信息批量导入失败

**问题**：尝试通过Excel批量导入员工信息时失败。

**解决方案**：
1. 下载系统提供的导入模板，确保使用正确的格式
2. 检查导入数据是否符合系统要求，特别是日期、身份证号等格式
3. 确保Excel文件不包含合并单元格或特殊格式
4. 检查必填列是否都已填写
5. 尝试分批导入，减少单次导入的数据量
6. 查看导入失败提示，根据提示修正数据后重新导入

### 3.3 员工离职操作问题

**问题**：办理员工离职后，相关数据处理不正确。

**解决方案**：
1. 确认离职日期设置是否正确
2. 检查是否已完成工作交接手续
3. 离职操作后，系统会自动将员工状态变更为「已离职」，相关权限会被取消
4. 如需要重新启用已离职员工，请联系系统管理员进行特殊处理

## 4️⃣ 考勤管理问题

### 4.1 打卡记录异常

**问题**：打卡后系统未显示打卡记录或显示错误。

**解决方案**：
1. 检查打卡时间是否在规定范围内
2. 确认打卡地点是否符合要求（如有地点限制）
3. 检查网络连接是否正常
4. 尝试重新打卡
5. 如问题持续，可提交补卡申请

### 4.2 补卡申请流程

**问题**：忘记打卡或打卡异常，需要申请补卡。

**解决方案**：
1. 从左侧菜单选择「考勤管理」→「打卡记录」
2. 找到需要补卡的日期，点击「补卡」按钮
3. 填写补卡原因、实际打卡时间
4. 上传相关证明材料
5. 提交申请等待审批
6. 审批结果可在「补卡记录」中查看

### 4.3 请假申请被拒绝

**问题**：提交的请假申请被审批人拒绝。

**解决方案**：
1. 查看审批意见，了解拒绝原因
2. 根据拒绝原因调整请假信息（如更改请假时间、提供更多证明材料等）
3. 重新提交请假申请
4. 如对审批结果有异议，可直接与审批人沟通

### 4.4 考勤统计数据不准确

**问题**：考勤统计页面显示的数据与实际情况不符。

**解决方案**：
1. 检查考勤规则设置是否正确
2. 确认排班设置是否准确
3. 核对打卡记录是否完整
4. 检查是否有待审批的补卡或请假申请
5. 如确认数据有误，请联系系统管理员处理

## 5️⃣ 培训管理问题

### 5.1 培训报名问题

**问题**：无法报名参加培训课程。

**解决方案**：
1. 确认课程报名时间是否已截止
2. 检查您是否符合课程参加条件
3. 确认课程是否已满员
4. 如以上情况都不适用，请联系培训管理员

### 5.2 培训证书查询

**问题**：完成培训后，如何查询或获取培训证书。

**解决方案**：
1. 从左侧菜单选择「培训管理」→「培训证书」
2. 在证书列表中查询您的证书
3. 可点击「查看」或「下载」按钮获取证书电子版
4. 如证书尚未生成，请联系培训管理员确认证书颁发时间

### 5.3 培训记录不显示

**问题**：参加培训后，系统中没有显示相关培训记录。

**解决方案**：
1. 确认培训是否已被培训管理员记录到系统中
2. 检查培训日期范围，确保查询条件正确
3. 联系培训管理员核实培训记录是否已录入

## 6️⃣ 资质证书管理问题

### 6.1 证书上传审核时间

**问题**：上传证书后，审核需要多长时间？

**解决方案**：
1. 证书审核通常在3个工作日内完成
2. 紧急情况可联系证书管理员加速处理
3. 审核结果会通过系统通知或邮件告知

### 6.2 证书即将到期提醒

**问题**：如何设置证书到期提醒？

**解决方案**：
1. 系统会自动对所有证书进行有效期监控
2. 证书到期前30天、15天、7天会自动发送提醒通知
3. 您可以在系统首页或通知中心查看即将到期的证书信息
4. 如需为特定证书设置自定义提醒，请联系系统管理员

### 6.3 证书信息修改

**问题**：上传的证书信息有误，需要修改。

**解决方案**：
1. 如证书尚未审核，可直接在证书管理页面找到该证书，点击「编辑」按钮修改
2. 如证书已审核通过，需要联系证书管理员进行修改
3. 如需更换证书电子版，可重新上传新版本并申请更新

## 7️⃣ GMP合规性管理问题

### 7.1 权限不足

**问题**：尝试访问某些功能时，系统提示权限不足。

**解决方案**：
1. 确认您的用户角色是否包含该功能的访问权限
2. 检查您的资质证书是否有效（某些GMP相关操作需要特定证书）
3. 如需临时获取权限，请联系部门负责人或系统管理员申请权限提升

### 7.2 电子签名失败

**问题**：进行电子签名时提示签名失败。

**解决方案**：
1. 确认您的用户名和密码输入是否正确
2. 检查您是否有权限进行此项操作的电子签名
3. 确认当前操作是否需要电子签名
4. 如签名功能异常，请联系系统管理员

### 7.3 审计日志查询

**问题**：如何查询自己的操作审计日志？

**解决方案**：
1. 从左侧菜单选择「GMP合规性管理」→「审计日志」
2. 在搜索条件中输入您的用户名
3. 设置日期范围
4. 点击「查询」按钮
5. 系统将显示您的所有操作记录

### 7.4 GMP活动追溯查询

**问题**：如何查询特定GMP活动的参与记录？

**解决方案**：
1. 从左侧菜单选择「GMP合规性管理」→「GMP活动追溯」
2. 设置查询条件，如活动类型、日期范围、参与人员等
3. 点击「查询」按钮
4. 可查看活动详情和所有参与人员记录

## 8️⃣ 系统设置问题

### 8.1 个人信息修改

**问题**：如何修改个人联系信息或其他基本资料？

**解决方案**：
1. 点击顶部导航栏的用户头像
2. 选择「个人设置」选项
3. 在基本信息页面修改相关信息
4. 点击「保存」按钮
5. 某些敏感信息（如身份证号）可能需要管理员审核才能修改

### 8.2 系统通知设置

**问题**：如何设置或管理系统通知？

**解决方案**：
1. 点击顶部导航栏的通知图标
2. 选择「通知设置」
3. 您可以设置接收通知的类型和方式
4. 可以标记通知为已读或删除通知

### 8.3 界面显示问题

**问题**：系统界面显示不完整或布局错乱。

**解决方案**：
1. 检查浏览器缩放比例是否为100%
2. 清除浏览器缓存和Cookie
3. 更新浏览器至最新版本
4. 尝试使用不同的浏览器
5. 调整屏幕分辨率

## 9️⃣ 故障排除

### 9.1 系统无响应

**问题**：系统突然无响应，无法进行任何操作。

**解决方案**：
1. 刷新页面（按F5或Ctrl+R）
2. 关闭当前浏览器窗口，重新打开并登录
3. 检查网络连接是否正常
4. 等待几分钟后重新尝试登录
5. 如问题持续，请联系系统管理员报告故障

### 9.2 数据丢失

**问题**：输入的数据意外丢失或未保存。

**解决方案**：
1. 系统通常会自动保存草稿，但建议定期手动保存
2. 检查网络连接，确保数据传输正常
3. 如数据确实丢失，可尝试恢复最近的备份
4. 联系系统管理员寻求帮助

### 9.3 错误提示

**问题**：系统显示错误提示信息。

**解决方案**：
1. 仔细阅读错误提示内容，根据提示进行操作
2. 记录错误信息，便于向技术支持人员描述问题
3. 尝试刷新页面或重新登录
4. 如无法自行解决，请联系系统管理员并提供错误信息
