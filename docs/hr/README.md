# 人力资源管理系统文档总览

## 1. 子系统概述

人力资源管理系统(HR-Management)是GMP系统的重要组成部分，负责企业人力资源相关信息的管理，包括员工信息管理、组织架构管理、招聘管理、培训管理、绩效管理、薪酬福利管理等功能。该系统确保人员管理满足GMP合规要求，特别是在人员资质、培训记录和权限管理方面提供全面支持。

## 3. 核心功能模块

### 3.1 员工信息管理
- 员工基本信息维护
- 员工入职、调岗、离职流程管理
- 员工资质与证书管理
- 员工档案电子化管理

- 部门层级结构管理

- 汇报关系维护
- 组织结构图表生成

### 3.3 招聘管理
- 招聘需求提报与审批
- 候选人信息管理
- 面试流程管理
- 入职流程自动触发

### 3.4 培训管理
- 培训计划制定与发布
- 培训课程管理
- 培训记录追踪
- 培训效果评估
- 培训证书管理

### 3.5 绩效管理
- 绩效目标设定
- 定期绩效考核
- 绩效评估结果记录
- 绩效数据分析与报表

### 3.6 薪酬福利管理
- 薪资结构管理
- 工资核算与发放记录
- 福利项目管理
- 社保公积金管理

### 3.7 考勤管理
- 考勤规则设置
- 打卡记录管理
- 请假加班申请审批
- 考勤异常处理

### 3.8 GMP合规管理
- 关键岗位人员资质审查
- 培训计划合规性检查
- 人员变更影响评估
- 权限与角色管理（与AUTH-SYS集成）

## 4. 技术架构

### 4.1 技术栈
- **后端框架**：Spring Boot、Spring Cloud
- **数据库**：PostgreSQL
- **缓存**：Redis
- **认证授权**：与AUTH-SYS集成，使用JWT
- **消息队列**：RabbitMQ（用于异步通知）
- **文档生成**：Apache POI
- **搜索引擎**：Elasticsearch（用于员工信息检索）

### 4.2 系统架构
- 采用微服务架构设计
- 前后端分离模式
- RESTful API接口
- 事件驱动的异步处理
- 与其他GMP子系统紧密集成

## 5. 关键业务流程

### 5.1 员工入职流程
从招聘系统获取候选人信息，完成入职登记，自动创建员工账号，分配初始权限，触发入职培训流程，并记录所有操作。

### 5.2 培训管理流程
培训需求收集→培训计划制定→课程开发→培训执行→培训记录→效果评估→证书颁发，全流程可追溯，确保符合GMP培训要求。

### 5.3 人员资质管理流程
资质录入→资质审核→有效期监控→提醒更新→变更记录，确保关键岗位人员资质持续有效。

## 6. 系统接口

### 6.1 内部接口
- **与认证授权系统(AUTH-SYS)**：用户账号创建、权限分配、身份验证
- **与培训管理系统(TRAINING-MANAGEMENT)**：培训需求传递、培训记录同步
- **与质量管理系统(QMS)**：人员资质验证、培训合规性检查
- **与制造执行系统(MES)**：操作人员权限校验
- **与消息服务(MESSAGE-SERVICE)**：通知消息发送
