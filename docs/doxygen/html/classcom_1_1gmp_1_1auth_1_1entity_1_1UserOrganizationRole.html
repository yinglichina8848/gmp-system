Detailed Description

GMP系统用户组织角色关联实体

实现用户、组织、角色三者间的关联管理，支持：

  • 用户可同时拥有多个组织内的不同角色
  • 临时权限分配和自动过期
  • 权限分配的审批流程
  • 完整的权限变更历史记录
  • GMP合规的权限追溯要求
