Detailed Description

GMP系统组织架构实体

支持多层次组织架构管理，包括：

  • 部门管理架构：行政、人事、财务等职能部门
  • 生产管理架构：车间、分车间、生产线级别的生产组织
  • 质量管理架构：质量检验、审核、体系管理等
  • 合规审核架构：GMP审核、法规事务等
  • 培训管理架构：培训规划、执行、评估等
