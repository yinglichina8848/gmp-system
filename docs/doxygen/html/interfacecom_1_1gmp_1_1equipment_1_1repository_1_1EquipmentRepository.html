Detailed Description

设备数据仓库接口

提供设备管理的完整数据访问层功能：

  • 基础CRUD操作（继承自JpaRepository）
  • 设备状态查询
  • 维护计划查询
  • 统计聚合查询
  • 业务规则验证方法
