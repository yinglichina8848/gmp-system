Detailed Description

GMP设备台账实体

设备实体类 表示设备的基本信息和状态

管理制药生产设备的完整生命周期，包括：

  • 设备基本信息和技术参数
  • 采购、安装、验收记录
  • 运行状态和使用监控
  • 维护计划和历史记录
  • 校准验证和合格状态
  • 财务成本和折旧信息
