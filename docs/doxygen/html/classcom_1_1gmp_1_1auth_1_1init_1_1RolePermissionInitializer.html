Detailed Description

GMP系统角色权限初始化器

负责在系统启动时初始化角色和权限，包括:

  • 创建预设角色
  • 创建系统权限
  • 建立角色与权限的关联

实现CommandLineRunner接口，确保系统启动时自动执行初始化
