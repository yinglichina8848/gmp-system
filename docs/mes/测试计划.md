## 概述

### 文档目的
本文档定义了GMP系统MES制造执行子系统的测试计划，包括测试范围、策略、环境、用例和进度安排等内容，确保MES系统符合GMP法规要求并满足业务需求。

### 参考文档
- [MES系统总体需求](总体需求.md)
- [MES系统架构设计](架构设计.md)
- [MES系统详细需求](详细需求.md)
- [MES系统用户故事](用例与用户故事.md)

### GMP特殊要求
- **合规性验证**: 所有功能必须符合GMP法规要求
- **审计可追溯**: 关键操作必须支持完整审计追踪
- **数据完整性**: 生产相关信息必须准确记录和存储
- **权限分离**: 生产操作与审核必须实施职责分离原则

## 测试范围

### 包含的功能模块
1. **生产订单管理模块**
   - 生产订单创建与下达
   - 生产订单状态跟踪
   - 生产订单完成确认
   
2. **工艺流程管理模块**
   - 工艺路线定义与管理
   - 工序参数设置与调整
   - 工艺版本控制
   
3. **生产执行管理模块**
   - 生产任务分配与执行
   - 生产进度实时跟踪
   - 生产异常处理
   
4. **物料批次管理模块**
   - 原辅料批次管理
   - 中间产品批次管理
   - 成品批次管理
   
5. **生产数据采集模块**
   - 生产参数自动采集
   - 质量数据录入与确认
   - 生产报表生成

### 不包含的内容
- 与硬件设备的直接驱动接口
- 复杂的APS高级计划排程功能
- 实时SCADA监控大屏

## 测试策略

### 测试类型
1. **单元测试**
   - 验证各服务类的方法逻辑正确性
   - 覆盖率目标：85%以上
   
2. **集成测试**
   - 验证各模块间的接口调用正确性
   - 验证数据库操作正确性
   
3. **接口测试**
   - 验证RESTful API的功能正确性
   - 验证请求/响应数据格式
   
4. **业务流程测试**
   - 验证完整生产订单执行流程
   - 验证物料批次追溯流程
   - 验证生产异常处理流程
   
5. **安全测试**
   - 验证数据访问权限控制
   - 验证生产数据完整性保护
   
6. **性能测试**
   - 验证大数据量生产记录查询性能
   - 验证并发生产操作处理能力

### 测试方法
- **黑盒测试**: 验证功能是否符合需求
- **白盒测试**: 验证代码逻辑是否正确
- **灰盒测试**: 结合前后端验证业务流程

## 测试用例

### 功能测试用例

#### TC001: 生产订单创建与下达
- **前置条件**: 用户具有生产计划员权限
- **测试步骤**:
  1. 登录系统，进入生产订单管理模块
  2. 点击"新建生产订单"按钮
  3. 填写产品信息、数量、计划时间等
  4. 选择合适的工艺路线
  5. 提交并下达生产订单
- **预期结果**: 
  1. 生产订单成功创建并分配编号
  2. 订单状态变为"已下达"
  3. 相关生产部门收到通知
  4. 操作日志完整记录

#### TC002: 物料批次追溯
- **前置条件**: 存在已完成的生产订单
- **测试步骤**:
  1. 登录系统，进入批次追溯模块
  2. 输入成品批次号进行查询
  3. 查看该批次产品的完整物料清单
  4. 追溯所用原辅料的来源批次
- **预期结果**: 
  1. 成功显示成品批次信息
  2. 完整展示物料清单结构
  3. 可追溯到所有原辅料批次
  4. 追溯信息完整准确

#### TC003: 生产异常处理
- **前置条件**: 存在正在进行的生产订单
- **测试步骤**:
  1. 在生产过程中录入异常信息
  2. 提交异常报告并指定处理人
  3. 处理人接收异常并进行处理
  4. QA确认异常处理结果
- **预期结果**: 
  1. 异常及时记录并通知相关人员
  2. 处理流程按照规定路径流转
  3. 处理结果得到QA确认
  4. 异常记录完整保存

### 安全测试用例

#### TC004: 生产数据完整性
- **前置条件**: 系统中存在生产记录数据
- **测试步骤**:
  1. 尝试直接修改数据库中生产记录
  2. 查询相关生产数据
- **预期结果**: 
  1. 数据库修改被拒绝或触发告警
  2. 系统数据保持一致和完整

#### TC005: 权限分级控制
- **前置条件**: 不同角色的用户账户
- **测试步骤**:
  1. 使用操作工账号尝试修改工艺参数
  2. 使用质检员账号尝试确认未检验批次
- **预期结果**: 
  1. 系统拒绝越权操作
  2. 提示相应权限不足信息
