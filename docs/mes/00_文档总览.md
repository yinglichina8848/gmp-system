# MES子系统文档总览

## 子系统概述

制造执行系统(MES)是GMP信息管理系统的核心组成部分，负责管理生产批次、设备监控、操作执行等关键制造流程。MES子系统提供了完整的生产过程控制和监控功能，确保生产活动符合GMP标准要求。

## 主要功能模块

- **批次管理**：生产批次的创建、启动、完成、暂停等全生命周期管理
- **设备监控**：实时监控设备状态、性能指标和告警管理
- **操作执行**：管理批操作流程，确保操作按标准程序执行
- **生产订单关联**：与生产订单系统集成，协调生产计划执行

## 技术栈

- **后端框架**：Spring Boot
- **数据访问**：Spring Data JPA
- **数据库**：支持关系型数据库
- **缓存**：Redis
- **文档工具**：Doxygen
- **测试框架**：JUnit 5
- **覆盖率工具**：JaCoCo

## 开发指南

### 环境要求

- JDK 17+
- Maven 3.8+
- MySQL/PostgreSQL
- Redis

### 构建和运行

```bash
# 构建项目
mvn clean package

# 运行测试
mvn test

# 生成API文档
doxygen

# 生成覆盖率报告
mvn jacoco:report
```
