# LIMS子系统（实验室管理系统）用例与用户故事

## 1. LIMS核心功能用例

### 1.1 样品测试流程用例

#### 用例ID：UC-LIMS-001
#### 用例名称：样品接收与测试任务分配
#### 参与者：样品接收员、实验室管理员、检验人员
#### 前置条件：样品已从生产车间或仓库提交
#### 基本流程：
1. 样品接收员接收样品并验证样品信息
2. 在LIMS系统中登记样品基本信息：
   - 样品编号（自动生成）
   - 样品类型和来源
   - 接收日期和保存条件
   - 关联的生产批次信息
3. 系统自动分配测试任务到相应实验室
4. 实验室管理员审核测试任务分配
5. 检验人员接收测试任务并确认测试方法
6. 系统生成测试工作清单，指导检验操作

#### 验收标准：
- [ ] 样品登记表单完整且易用
- [ ] 测试任务自动分配准确
- [ ] 工作清单清晰明确
- [ ] 权限控制严格

### 1.2 测试结果审核用例

#### 用例ID：UC-LIMS-002
#### 用例名称：测试结果录入与审核发布
#### 参与者：检验人员、审核人员、质量管理人员
#### 前置条件：测试任务已分配，仪器设备就绪
#### 基本流程：
1. 检验人员在系统中查看待执行的测试任务
2. 系统显示测试方法和标准操作规程
3. 检验人员记录测试过程和使用仪器信息
4. 系统自动采集或手动录入测试结果数据
5. 【中药特色】对于指纹图谱、多指标成分等数据，系统提供专门的分析工具
6. 系统基于设定的标准自动判断测试结果状态
7. 检验人员提交测试结果进行审核
8. 审核人员审核测试数据完整性和准确性
9. 审核通过后，测试结果正式发布并关联产品批次

#### 验收标准：
- [x] 测试数据录入友好便捷
- [x] 自动判断功能准确可靠
- [x] 审核流程规范完整
- [x] 结果发布及时有效
- [x] 支持中药特色数据分析功能

### 1.3 中药材鉴别用例

#### 用例ID：UC-LIMS-003
#### 用例名称：中药材传统鉴别与指纹图谱分析
#### 参与者：中药材鉴别专家、检验人员
#### 前置条件：
- 样品已接收并登记
- 检验人员具备中药材鉴别资质
#### 基本流程：
1. 检验人员选择待鉴别的中药材样品
2. 系统显示该药材的标准性状特征和显微特征
3. 检验人员进行性状鉴别，记录特征描述
4. 系统提供拍照功能，采集药材外观照片
5. 检验人员进行显微鉴别，记录显微特征
6. 系统提供显微照片采集功能
7. 检验人员选择指纹图谱分析方法
8. 系统引导完成样品前处理和仪器分析
9. 系统采集并存储指纹图谱数据
10. 系统自动与标准图谱进行比对分析
11. 检验人员根据鉴别结果和图谱分析形成最终鉴别结论
12. 鉴别结果提交审核

#### 验收标准：
- [x] 支持多种传统鉴别方法记录
- [x] 图片采集和对比分析功能正常
- [x] 指纹图谱数据采集和分析准确
- [x] 与标准图谱比对功能有效
