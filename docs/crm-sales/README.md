# CRM销售管理子系统文档

本目录包含CRM销售管理子系统的完整文档，涵盖需求分析、系统设计、实现方案、测试用例以及用户指南等内容。

## 文档结构

- **根目录**：包含文档总览、需求分析、架构设计等核心文档
- **用户需求**：用户需求相关文档
- **系统软件自身需求**：系统软件需求规约和非功能性需求
- **设计**：详细设计、数据库设计、API设计等
- **实现**：实现文档和代码规范
- **测试**：各类测试文档
- **用户文档**：用户手册和管理员手册
- **示例**：使用示例
- **项目进展**：项目计划和项目日志
