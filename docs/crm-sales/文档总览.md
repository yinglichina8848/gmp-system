# CRM销售系统文档总览

## 1. 子系统概述

CRM销售系统（CRM-SALES）是GMP系统中的一个重要组成部分，专注于管理客户关系和销售流程。该系统负责记录和跟踪与客户的所有交互，管理销售线索和商机，执行销售预测和分析，确保销售流程的合规性和可追溯性，为企业提供完整的客户生命周期管理能力。

## 4. 技术架构

### 4.1 技术栈
- 后端：Java 17, Spring Boot 3.x, Spring Cloud
- 数据库：PostgreSQL 14
- 缓存：Redis
- 消息队列：RabbitMQ
- 前端：React, Ant Design Pro
- API网关：Spring Cloud Gateway
- 服务注册与发现：Nacos

### 4.2 架构设计
- 采用微服务架构设计
- 前后端分离架构
- 服务间通过RESTful API和消息队列通信
- 数据库采用主从复制确保数据可靠性
- 支持水平扩展以应对高并发场景

### 4.3 安全架构
- 基于OAuth 2.0和JWT的认证授权
- API访问权限控制
- 敏感数据加密存储
- 审计日志记录所有关键操作
- 防SQL注入和XSS攻击
