# 移动应用文档总览

## 1. 子系统概述

移动应用（MOBILE-APP）是GMP系统的移动端支持组件，为用户提供便捷的移动访问能力。该应用支持iOS和Android两大主流平台，让用户能够随时随地访问GMP系统的核心功能，提高工作效率和响应速度，特别是在生产现场和仓库管理等场景中发挥重要作用。

## 4. 技术架构

### 4.1 技术栈
- 前端框架：Flutter 3.x（跨平台开发）
- 原生集成：iOS Swift/Objective-C，Android Kotlin/Java
- 后端通信：RESTful API, WebSocket
- 数据存储：SQLite, SharedPreferences, Keychain
- 身份认证：OAuth 2.0, JWT
- 地图服务：Google Maps/高德地图
- 推送服务：Firebase Cloud Messaging, 极光推送

### 4.2 架构设计
- MVVM架构模式
- 响应式编程
- 模块化设计
- 离线工作能力
- 数据同步机制

### 4.3 安全架构
- 传输层安全（TLS 1.3）
- 应用级加密
- 安全存储
- 防截屏/录屏保护
- 敏感操作二次验证
